## オフィスチェア・デスクチェアのサイズの選び方

オフィスチェアやデスクチェアを選ぶ際に、多くの人が悩むのがサイズです。「自分に合ったサイズはどれだろう？」「正しい座面高は？」といった疑問は、快適な作業環境を求める上で非常に重要です。ここでは、身長別の最適な座面高やデスクとの距離、アームレストの位置、座面の幅・奥行きについて詳しく解説します。

### 身長別の最適な座面高

座面の高さは、椅子の快適さに直結しています。座面高が合わないと、血行が悪くなったり、疲れやすくなったりします。以下の表は、身長別に推奨される座面高の目安を示しています。

| 身長      | 推奨座面高 (cm) |
|-----------|-----------------|
| 150cm以下 | 40〜45          |
| 150〜160cm| 45〜50          |
| 160〜170cm| 50〜55          |
| 170〜180cm| 55〜60          |
| 180cm以上 | 60〜65          |

この表を参考に、自分の身長に合った座面高を選ぶことが重要です。座面高は調整可能な椅子も多いため、調整機能がある椅子を選ぶとさらに便利です。

### デスクとの距離の計算方法

オフィスチェアを選ぶ際、デスクとの距離も考慮する必要があります。一般的に、座面高とデスクの高さの差は約25〜30cmが理想です。この差が小さすぎると、膝がデスクにぶつかってしまい、大きすぎると腕が上がりすぎてしまいます。

以下の計算式を使って、デスクとの適切な高さの差を確認してみましょう。

```
デスク高さ - 座面高 = 理想的な差（25〜30cm）
```

例えば、デスクの高さが75cmの場合、座面高は以下のように計算できます。

- 最低座面高: 75cm - 30cm = 45cm
- 最高座面高: 75cm - 25cm = 50cm

この場合、座面高は45〜50cmの範囲が理想的です。

### アームレストとの干渉に注意するポイント

アームレストがある椅子を選ぶ場合、デスクとの干渉についても考慮する必要があります。アームレストがデスクに当たらないためには、アームレストの高さが重要です。以下のポイントを確認しましょう。

1. **アームレストの高さ**: デスクの高さに対してアームレストが高すぎないか確認します。理想的には、アームレストがデスクよりも低いことが望ましいです。
2. **アームレストの幅**: アームレストの幅がデスクの幅に対して適切であるかも考慮しましょう。デスクに座った際に、アームレストがデスクにぶつからないことが大切です。

### 座面の幅・奥行きの選び方

座面の幅と奥行きも、快適さに大きく影響します。以下の基準を参考に、適切なサイズを選びましょう。

- **座面幅**: 一般的には45〜50cm程度が多いですが、体格に応じて選ぶことが重要です。座ったときに左右に少し余裕があると良いでしょう。
- **座面奥行き**: お尻から背中までの長さに応じて、奥行きを選びます。一般的には40〜50cmが推奨されますが、背もたれとの距離も考慮してください。

以下の表は、座面幅と奥行きの目安を示しています。

| サイズ      | 幅 (cm) | 奥行き (cm) |
|-------------|---------|-------------|
| スタンダード | 45〜50  | 40〜45      |
| 幅広タイプ   | 50〜55  | 45〜50      |
| コンパクト   | 40〜45  | 35〜40      |

自分の体格や好みに合わせて、最適なサイズを見つけましょう。

### まとめ

オフィスチェアやデスクチェアを選ぶ際のポイントを以下にまとめます。

- **身長に応じた座面高を選ぶ**: 身長別の推奨座面高を参考に、自分に合ったサイズを選びましょう。
- **デスクとの高さの差を確認**: デスクの高さと座面高の差は25〜30cmが理想です。
- **アームレストの干渉に注意**: アームレストの高さと幅がデスクと干渉しないよう確認します。
- **座面の幅・奥行きを選ぶ**: 体格に応じて、座面の幅と奥行きを選ぶことが重要です。
